# Assessment-Repo

A full-stack employee management web application built with **ASP.NET Core Minimal API** (.NET backend) and **React + TypeScript** (frontend), deployed on a shared IIS host under a single domain.

Live demo: [https://wilmar-assessment.premiumasp.net/wilmar-assessment/](https://wilmar-assessment.premiumasp.net/wilmar-assessment/)

---

## Tech Stack

### Backend вЂ” `BE-.NET`
- **ASP.NET Core** вЂ” Minimal API
- **Entity Framework Core** вЂ” ORM with SQL Server
- **Dapper** вЂ” for raw/stored procedure queries
- **Rate Limiting** вЂ” built-in ASP.NET Core rate limiter
- **CORS** вЂ” configured for cross-origin frontend requests

### Frontend вЂ” `FE-React`
- **React 18 + TypeScript**
- **Vite** вЂ” build tool
- **TanStack Query v5** вЂ” server state, caching, and mutations
- **React Hook Form + Zod** вЂ” form handling and schema validation
- **shadcn/ui + Tailwind CSS** вЂ” UI components and styling
- **Sonner** вЂ” toast notifications

---

## Features

- **Employee List** вЂ” paginated table with sort (Name, BirthDate, Sex, Salary) and order (Ascending/Descending) controls
- **Create Employee** вЂ” form dialog with validation (name, NIK, birth date, salary, address, sex)
- **Edit Employee** вЂ” pre-filled form dialog; NIK is read-only on edit
- **Delete Employee** вЂ” confirmation dialog before deletion
- **OCR endpoint** вЂ” additional API route for OCR processing -> only in POSTMAN
- **Global error handling** вЂ” API errors surface as toast notifications via Sonner

## Programmability - Stored Procedure

For pagination Employee
```
CREATE PROCEDURE [dbo].[GetEmployees]

    @PageNumber INT = 1,
    @PageSize INT = 30,
    @SortBy NVARCHAR(50) = 'Id',
    @IsDesc BIT = 0

AS
BEGIN

    SET NOCOUNT ON;

    DECLARE @Offset INT = (@PageNumber - 1) * @PageSize;
    DECLARE @SQL NVARCHAR(MAX);

    SET @SQL = N'
        SELECT
            *,
        COUNT(*) OVER() AS TotalCount
        FROM dbo.Employee
        ORDER BY ' + @SortBy + 
            CASE 
                WHEN @IsDesc = 1 THEN ' DESC' ELSE ' ASC' 
                END + N'
        OFFSET @Offset ROWS FETCH NEXT @PageSize ROWS ONLY;
    ';

    EXEC sp_executesql @SQL,
        N'@Offset INT, @PageSize INT',
        @Offset, @PageSize;
END
```
